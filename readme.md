# readme 文档
- 项目说明文档
- 一般都与项目放一起

## git操作
- git init
- 初始化成功后，当前目录后面有(master)

## 工作区
- 持有实际文件
- 我们平时增删改查的文件都是工作区的内容

## 暂存区
- 本地仓库的一个主要组成部分
- 可以理解为我们看不到的一个地方

## 本地仓库
- 可以理解为我们看不到的一个地方
- 也是存在于用户电脑中的
- 用于存储项目及版本项目记录等信息

## 提交到暂存区
- git add 文件名
- 将工作区的变动提交到暂存区
- git add .  就是把所有变动

## 查看工作区和暂存区的状态
- git status (多使用、查看)
- 查看增删改
